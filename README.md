# LetsPlayMC

**LetsPlayMC** is a desktop Minecraft server manager providing an easy-to-use graphical interface to create, start, and manage Minecraft servers. Designed with features for players and admins alike, LetsPlayMC simplifies server management tasks and enhances the multiplayer experience.

---
## 🚀 Features

- **Server Management**:
  - Create, start, stop, restart, or delete Minecraft servers effortlessly.
  - Manage server configuration files.

- **Player Management**:
  - View online player details.
  - Inspect player data, including in-game inventory.

- **Enhanced Addon Support**:
  - Search and install Minecraft addons from the Modrinth repository.
  - Easily uninstall and manage installed addons.
  - Display addon metadata (author, version, downloads).

- **Cross-Platform Compatibility**: Available on Windows, macOS, and Linux.

## 🛠 Installation

To get started, follow these steps:

1. **Prerequisites**:
   - Python 3.9.6 (or higher).
   - pip (Python package manager).

2. **Clone the Repository**:
   ```bash
   git clone https://github.com/yourusername/letsplaymc.git
   cd letsplaymc
   ```

3. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run LetsPlayMC**:
   ```bash
   python src/main.py
   ```

---
## 📘 Usage

- **Starting a Server**: Navigate to the server tab, click create, and configure the desired options.
- **Players Tab**: View online players and inspect player inventories.
- **Addons Tab**: Search for new addons, view details, and install/uninstall them on your server.
